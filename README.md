# ChobiLibCS

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

ChobiLibCS is a collection of pure C# utilities and core logic for chobitech's projects.

## Overview
This library is primarily designed for internal use within chobitech's Unity packages (e.g., MyBrand.Core). It contains engine-agnostic logic to ensure high performance and reusability across different environments.

## Status
- **Internal Use**: Documentation is not provided as this is a private-utility-first library.
- **Support**: Limited. Please use at your own risk.
